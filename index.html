What This System Is

AADP runs on a Raspberry Pi 5 (16GB) in Bill’s home. It’s always on. It runs ~25 n8n workflow agents that monitor system health, ingest research, synthesize findings, and manage a growing library of operational lessons. Claude Code is the primary builder — it reads directives, executes them, writes session artifacts, and leaves the system more capable after every session.

The long-term goal: Bill gives the system a high-level intention and it autonomously researches, decomposes the work, builds capabilities, and executes — keeping Bill in the loop without requiring his approval at every step.
