What This System Is

AADP runs on a Raspberry Pi 5 (16GB). It’s always on. It runs 9 n8n workflow agents that monitor system health, ingest research, synthesize findings, and manage 237 operational lessons. Claude Code is the primary builder — it reads directives, executes them, writes session artifacts, and leaves the system more capable after every session.

The long-term goal: Bill gives a high-level intention and the system autonomously researches, decomposes the work, builds capabilities, and executes — keeping Bill in the loop without requiring approval at every step.
